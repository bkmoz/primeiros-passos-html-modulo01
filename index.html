O que é Linguagem de Programação?

(Voltar)

Linguagem de Programação: é um conjunto de instruções e regras usadas para criar programas (softwares) que são escritas de forma formal.

Essas instruções são usadas para controlar o comportamento do computador e criar aplicativos, jogos e outros software

Linguagem Server-Side ?

(Voltar)

Linguagem Server-side é qualquer linguagem de programação ou script executado no servidor web, em vez do navegador do usuário.

Os principais exemplos de linguagens de servidor são PHP, Ruby, JavaScript, C# e Java.

Linguagem Client-Side?

(Voltar)

Linguagem Client-side é um tipo de linguagem de programação usada para desenvolver aplicativos web que são executados no lado do cliente.Os clientes são geralmente navegadores (ou Browsers) da web, como o Mozilla Firefox, Google Chrome ou Explorer Edge., JavaScript também é uma linguagem client-side.

O que é HTML?

(Voltar)

HTML é o padrão de linguagem de marcação para documentos da Web. É usado para descrever a estrutura de um documento, como texto, imagens, links, etc. É um dos mais importantes padrões de linguagem da web, pois é a base para a maioria dos outros padrões e tecnologias.
